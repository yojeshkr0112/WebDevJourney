Escape Characters in HTML

Here are some common HTML escape character sequences:

  • < : &lt; (less than)
  • > : &gt; (greater than)
  • & : &amp; (ampersand)
  • " : &quot; (double quote)
  • ' : &apos; (single quote/apostrophe)
  • © : &copy; (copyright symbol)
  • ® : &reg; (registered trademark symbol)
  • € : &euro; (euro currency symbol)
